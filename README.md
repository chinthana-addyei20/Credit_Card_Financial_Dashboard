# Credit_Card_Financial_Dashboard
Project Overview
This project is an interactive Credit Card Analytics Dashboard built using Power BI to analyze customer behavior and credit card transaction data. The dashboard provides business insights through dynamic visualizations, KPIs, and filters, helping users understand revenue trends, customer demographics, spending patterns, and transaction performance.

Features
Interactive dashboard with slicers and filters
Revenue, Interest, Income, and Transaction analysis
Customer segmentation by:
Age Group
Gender
Education
Occupation
Marital Status
Income Group
Dependents 
Credit Card category analysis (Blue, Silver, Gold, Platinum)
Revenue analysis by expenditure type
Quarterly and weekly revenue trends
Customer and transaction performance comparison

Key KPIs
Total Revenue
Total Interest Earned
Total Income
Total Transaction Amount
Total Transaction Count
Customer Satisfaction Score (CSS)
